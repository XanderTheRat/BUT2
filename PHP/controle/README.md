**Options d'utilisation**
- Pour accéder au feed, vous devez posséder un compte. 
- Pour créer un compte, il vous suffit de cliquer sur "inscription"

- Une fois votre message envoyé, tout les utilisateurs y ont accès. Vous ne pouvez pas le modifier, mais il est possible de le liker. Un seul like par personne par message.
- Il est possible d'ajouter des commentaires pouvant etre votés. Il est possible de modifier son commentaire, mais il faudra alors tout supprimer ( y compris le nom d'utilisateur ) et réécrire votre  message.
- Vous pouvez voter les commentaires avec des upvotes et downvotes.
- Lorsqu'une personne vote un message ou un commentaire, une notification est envoyée. Un clique sur l'onglet de cloche permet de masquer toutes les notifications.
- L'icone de lune/soleil permet d'alterner entre le mode jour et nuit
